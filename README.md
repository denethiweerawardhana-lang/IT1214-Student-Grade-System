// Project Purpose
The Student Grade Management System is a Java console-based application designed for educational institutes to transition from manual spreadsheet records to a digital tracking system. Built using Object-Oriented Programming (OOP) principles, it allows teachers to manage academic records by adding student details, viewing records, searching for profiles via Student IDs, and calculating average marks.

// How to Run the Program

compile the java:
javac Main.java

Run:
java Main
